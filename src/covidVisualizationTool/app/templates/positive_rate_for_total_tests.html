{% block title %} How Did the Number of Tests Affect the Positivity Rate? {% endblock %}

Positivity Rate by Total Tests
The purpose of these charts is to answer the question of "Is the positivity rate higher when there are a higher number of new COVID-19 tests?" It is statistically evident that an increased number of tests also leads to an increased positivity rate. This could perhaps be attributed to asymptomatic individuals being required or encouraged to get tested. The number of tests are the number of tests performed. The figures are the sum across states, some of which may include serology tests in addition to PCR tests. Many commercial laboratories call it an antibody test. An antibody test looks for the presence of antibodies, which are specific proteins made in response to infections. (CDC) These charts only feature data from the United States.
