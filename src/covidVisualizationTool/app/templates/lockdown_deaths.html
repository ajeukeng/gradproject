{% block title %} How Did Strict Lockdowns Affect Deaths and Cases? {% endblock %}

{{ title }}
The purpose of these charts is to answer the question of "Did lockdowns help reduce deaths and cases?" The stringency index or the government response index is a composite measure based on 9 response indicators including school closures, workplace closures, and travel bans, rescaled to a value from 0 to 100, with 100 being the strictest response. In the chart below, the green shaded bubbles represent different countries. When hovered over, the stringency index and total deaths per million are shown for each country. The size of the bubbles represents the total number of cases for that country, to scale. The World Health Organization defines a confirmed case as “a person with laboratory confirmation of COVID-19 infection.”
