{% block title %} How Did the Number of People Vaccinated and Boosted Affect the Positivity Rate? {% endblock %}

The purpose of these charts is to answer the question of "Does the number of people partially vaccinated, fully vaccinated, and boosted affect the positivity test rate?" The number of tests are the number of tests performed. The figures are the sum across states, some of which may include serology tests in addition to Polymerase Chain Reaction (PCR) tests. Many commercial laboratories call serology tests an antibody test. An antibody test looks for the presence of antibodies, which are specific proteins made in response to infections. (CDC) The total number of COVID-19 vaccination booster doses administered include any doses administered beyond the number prescribed by the vaccination protocol, which is currently two Pfizer or Moderna shots or one Johnson & Johnson shot. These charts only feature data from the United States.
